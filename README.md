# LeadFlow CRM

Full Stack MERN Lead Management Dashboard.

## Features

- JWT Authentication
- Role Based Access
- CRUD Operations
- Search & Filters
- Pagination
- CSV Export
- Docker Support
- TailwindCSS UI

## Tech Stack

Frontend:
- React
- TypeScript
- TailwindCSS

Backend:
- Node.js
- Express
- MongoDB

## Setup

### Backend

cd backend
npm install
npm run dev

### Frontend

cd frontend
npm install
npm run dev

## Docker

docker compose up
